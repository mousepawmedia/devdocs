Glossary
#######################

..  glossary::

    absolute path
        A file path which specifies the location of the target from the
        root directory.

    backtrace
        A list of library and function calls that allows you to see how the
        problem line of code came to be called.

    portable
        (1) Works correctly on multiple computers, with little to no
        configuration beforehand.
        (2) Works on multiple operating systems.

    relative path
        A file path which specifies the location of the target *relative*
        to the current directory.
